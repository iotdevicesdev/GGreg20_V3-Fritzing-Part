# GGreg20_V3 Fritzing Part

Official Fritzing component for the **GGreg20_V3 Ionizing Radiation Detector** (Geiger Counter) module, manufactured by **IoT-devices**.

## Overview

This repository contains the official digital part for Fritzing, allowing engineers and makers to integrate the GGreg20_V3 module into their breadboard designs, electronic schematics, and PCB layouts.

### Features:
- **Breadboard View:** Realistic representation of the module.
- **Schematic View:** Clean and standard symbol for easy circuit design.
- **PCB View:** Accurately placed copper pads with correct 2.54mm (0.1") pitch and silk screen mask.
- **Full Metadata:** Proper family, manufacturer, and pin labeling.

## Installation

1. Download the latest `.fzpz` file.
2. Open Fritzing.
3. Go to **File > Open** and select the downloaded `.fzpz` file.
4. The GGreg20_V3 module will now appear in your **'Mine'** parts bin.

## Pinout

| Pin | Label | Description |
|-----|-------|-------------|
| 1   | BAT + | Power Supply (3.3V - 5V) |
| 2   | BAT - | Ground |
| 3   | OUT   | Pulse Output (Active Low) |
| 4   | GND   | Ground |
